**Task 1 – Landing Page**

This project is a simple and responsive landing page developed using HTML and CSS as part of the Web Development and Designing Internship at Oasis Infobyte.

The landing page demonstrates the basic structure of a modern webpage including sections such as header, content area, and footer. It focuses on layout design, alignment, color selection, spacing, and styling using CSS.
